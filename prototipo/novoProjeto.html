Sistema ERP Hospitalar Difícil

Prazo restante: 63 dias
Proposta: R$ 3500,00

Descrição

O freelancer deverá desenvolver um sistema ERP para uma rede de hospitais. Os hospitais contam com diversos setores, cada setor deverá possuir um subsistema próprio que atenda os requisitos necessitados. O sistema ERP deverá ser feito de forma que possa ser conectado com o sistema web já existente onde os pacientes podem marcar suas consultas.

Tempo máximo para produção: 1 ano
Oferta média: R$ 3000,00
